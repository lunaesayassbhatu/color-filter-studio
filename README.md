# Color Filter Studio

An interactive web app that applies real-time color filters and effects to photos using computer vision — built with Gradio and OpenCV.

## Filters

| Filter | Effect |
|--------|--------|
| 🌊 Ocean Blue | Boosts blue channel for a cool-tone look |
| 🌅 Sunset | Warms up reds and reduces blue for a golden feel |
| 🌿 Forest Green | Enhances greens for a natural, earthy tone |
| 🖤 Black & White | Converts to classic grayscale |
| ✨ Glitter | Adds sparkle noise for a glitter effect |
| 🔥 Emboss | Applies a 3D raised edge effect |
| 🌈 Invert Colors | Flips all color channels to their inverse |
| 💜 Purple Haze | Boosts red and blue for a dreamy purple look |

## Features

- Upload any photo and apply filters instantly
- Adjustable strength slider for fine-tuned control
- Dark-themed UI with a clean, modern layout
- Powered by OpenCV for fast image processing

## Usage

```bash
pip install -r requirements.txt
python app.py
```

Then open the local URL printed in the terminal to use the app in your browser.

## Tech Stack

- Python
- Gradio
- OpenCV
- NumPy
- Pillow
